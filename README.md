# Personal Portfolio Website

## Overview

This repository contains my personal portfolio website developed using HTML, CSS, and JavaScript. The website showcases my professional background, technical skills, QA automation experience, and project work in a structured and accessible format.

It serves as a central hub connecting my GitHub projects, automation frameworks, and contact information for recruiters and collaborators.
